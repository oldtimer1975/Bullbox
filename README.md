# BullBox – A csomagküldés jövője (közösségi, AI‑os, kriptós)

A BullBox egy globális, közösségi csomagküldő platform, ahol a feladók és a közösségi futárok egymásra találnak. A rendszer valós idejű GPS‑követést, beépített chatet, AI‑alapú dinamikus árazást, opcionális alkalmi biztosítást és kriptós/fiatos pénztárcát kínál. Cél: gyors, biztonságos, rugalmas kézbesítés, erős közösségi bizalommal.

Fő értékek:
- Gyorsaság és rugalmasság (közösségi futárhálózat)
- Biztonság (KYC, escrow, QR‑átadás, biztosítás)
- Pénzügyi szabadság (kriptó + fiat wallet)
- Okos élmény (AI‑asszisztens, dinamikus ár, push értesítések)

## MVP – Mit tud az első verzió?
- Globális indulás (HU+EN UI), webapp (reszponzív)
- Teljes KYC mindenkinek
- Kripto az MVP‑ben (non‑custodial), FIAT keretrendszer előkészítve (custodial)
- Teljes escrow + vitakezelés (admin)
- Csomagfeladás: méret/súly, címek, fotók, opcionális biztosítás
- Futár bejelentkezik és megad munkasugarat (pl. 3–5 km), elérhetőség
- Valós idejű GPS + QR‑os átadás/átvétel
- Beépített chat fotócsatolással + push értesítések
- AI‑árkalkuláció: km, futársűrűség és forgalom alapján (alkudozás chaten, véglegesítés gombbal)
- Kétirányú értékelés; profil stabilitási mutatók
- Referral rendszer (keret v1)
- GDPR megfelelés

## Roadmap (rövid)
1) SPEC és adatmodell rögzítés
2) Auth+KYC, User+Courier profil, radius beállítás
3) Package CRUD + biztosítás + fotók
4) AI árazás (baseline képlet + forgalom/futársűrűség faktor)
5) Claim + státuszgép + QR flow + GPS
6) Chat + push
7) Escrow + vitakezelés
8) Referral + profil metrikák
