# Branding assets

Place high-quality PNGs here (optional runtime images):
- bull-courier.png – courier hero used on Home and Courier Dashboard
- bull-head.png – brand head used on customer pages / header

SVGs used by default to avoid missing-asset errors:
- bull-courier.svg
- bull-head.svg

Make sure raw SVG files start with `<svg` as the first byte (no BOM / backticks).
