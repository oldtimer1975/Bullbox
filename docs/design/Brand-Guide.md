# BullBox – Brand Guide (v1)

## Logók és illusztrációk
- Brand mark: bika fej (ikon, avatar, favicon) – referencia: ![image1](image1)
- Hero illusztráció: biciklis bika (login/landing) – referencia: ![image2](image2)
- Export: SVG elsődlegesen; PNG fallback.
- Változatok: full, detailed, minimal (ikon 32×32)

## Színpaletta
- Primary: #E10600 (Bull Red) • Accent: #2563EB (Tech Blue)
- Background Dark: #0F172A • Foreground: #F8FAFC • Success/Warn/Error lásd tokens

## Tipográfia
- Headings: Inter Tight / Montserrat • Body: Inter • Tabuláris számok áraknál

## Használati irányelvek
- Kontraszt: WCAG AA • Ikon+szöveg együtt • Ne torzíts / ne változtass arányt
- Favicon/app‑icon: fej minimal
- Hero: biciklis illusztráció a bejelentkezésnél

## "Bull Anatomy" (onboarding)
- Szarvak = Biztonság; Szemek = Követés; Fülek = Értesítések; Száj = Chat/AI; Paták = Státuszok; Szív = Bizalom
